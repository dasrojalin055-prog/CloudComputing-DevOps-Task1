# Portfolio Website

## Project Description
This project is a simple responsive portfolio website created using HTML and CSS. It is developed as part of the Cloud Computing & DevOps Task.

## Features
- Responsive Portfolio Website
- Hero Section
- About Me Section
- Projects Section
- Contact Section
- Profile Image
- Clean and Simple Design

## Technologies Used
- HTML5
- CSS3

## Project Files
- index.html
- styles.css
- images/profile.jpg

## Deployment
This project will be hosted on AWS S3 Static Website Hosting and served using Amazon CloudFront.
